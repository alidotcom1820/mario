# Mario

Eine minimalistische Website für Video-Upload mit zufälligem Design und animiertem Text.

## Features

- Mobiloptimiert
- Video-Upload
- Zufälliges Design auf Videos
- Bouncing Text Animation ("designed by mario maserati")
- Intro Screen mit "desjash production"

## Lokale Entwicklung

Öffne `index.html` in einem Browser.

## Deployment auf Cloudflare

1. Erstelle ein Konto bei Cloudflare.
2. Gehe zu Cloudflare Pages.
3. Verbinde dein GitHub-Repository.
4. Deploye die statischen Dateien.
